## How to define a Class
    With the word class, followed by the name of it and brackets({}).

## How to add methods to a class
    By declaring a function inside of the class.

## Why and how to add a static method to a class
    With the word static followed by the name of the static method.
    It's useful for datas that doesn't need to be replicate in the instances.

## How to extend a class from another
    With the word extend.
    Example:
        class Dog extends Animal(mother class) {}
