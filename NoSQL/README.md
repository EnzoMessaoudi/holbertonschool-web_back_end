## What NoSQL means  
    It means Not Only SQL

## What is difference between SQL and NoSQL  
    In SQL, we use a database with columns, tables, rows or scehmas to organize and retrieve datas. Meanwhile, in NoSQL, it doesn't contains all of this, it used more flexible data models.
    NoSQL is prefered for stocking large amounts of datas, like messages, chats, log datas or images/videos.

## What is ACID  
    Atomicity Consistency Isolation Durability

## What is a document storage  
    It stores self-describing JSON, XML and BSON documents.

## What are NoSQL types  
    Key-Value data stores: Store value of any type  
    Document Stores: Used for self-describing JSON, XML and BSON documents  
    Wide-columns stores: Stores datas inside rows and columns BUT the names of the columns can vary depending from the rows  
    Graph stores: uses for storing map and query relationship.

## What are benefits of a NoSQL database  
    Scalability, Performences, High Availibility, Global Availibility and Flexible Data Modeling

## How to query information from a NoSQL database  
    Find (SELECT): Retrieve infos about a database.  

## How to insert/update/delete information from a NoSQL database  
    Use database: Command used to choose which database we use.  
    Find (SELECT): Retrieve infos about a database.  
    Insert (CREATE): Insert data inside of a database.  
    updateOne or updateMany: Uses to update database.  
    drop: Delete datas inside of a database.
