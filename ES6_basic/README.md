## What ES6 is  
    It's a language of programmation created to make easier to use JavaScript, to standardize it.

## New features introduced in ES6  
    let and const variable declaration, arrow functions, template literals and classes

## The difference between a constant and a variable  
    A variable can be modified later on. Meanwhile, a constant cannot be modified.  
    example:  
        let a = 1;
        const b = 2;

        a = 3; // Good !
        b = 4; // Error !
