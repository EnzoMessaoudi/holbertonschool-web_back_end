## What ES6 is  
    It's a language of programmation created to make easier to use JavaScript, to standardize it.

## New features introduced in ES6  
    let and const variable declaration, arrow functions, template literals and classes

## The difference between a constant and a variable  
    A variable can be modified later on. Meanwhile, a constant cannot be modified, it's a Read-only named constant.  
    Example:  
        let a = 1;  
        const b = 2;  

        a = 3; // Good !  
        b = 4; // Error !  

## Block-scoped variables  
    It's a variable which is declared inside of a block and can only ba access inside of it.  

## Arrow functions and function parameters default to them  
    It's a way to declare a function easier to read and faster to create.  
    Example:  
        const add = (a, b) => a + b;  
        console.log(add(5, 3)); // 8  

## Rest and spread function parameters  
    The spread operator expands elements of an array or object  
    The rest operator collects multiple values into an array  

## String templating in ES6  
    String templating in ES6 is used to write dynamic response in our code output.
    Example:
        console.log('Hello, ${1 + 1} is the answer!');
